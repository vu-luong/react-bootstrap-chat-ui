# React Bootstrap Chat UI

A collection of responsive UI components to build a chat application

<img src="./example/example.png">

## Getting Start

```bash
npm install react-bootstrap-chat-ui --save
```
or
```bash
yarn add react-bootstrap-chat-ui
```
